# ffmpeg-player
A macOS player base on ffmpeg using OC and Swift.

使用OC与Swift实现了基于ffmpeg的播放器，图像渲染,音频播放,音视频同步都是从0到1做了一个实现。图像渲染同时实现了OpenGL与Metal两种GPU渲染方式。
